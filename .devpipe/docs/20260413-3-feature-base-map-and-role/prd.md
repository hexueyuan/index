# 需求文档 (PRD)

## 基本信息

- 开发类型: 新功能
- 功能描述: 构建基础地图和操作角色
- GitHub Issue: #3
- 远程分支: main
- 本地分支: feature-base-map-and-role
- 创建时间: 2026-04-13

## 需求背景

当前项目（像素风个人主页游戏）已有基础框架：程序化生成的简易地图（纯色方块）和一个蓝色方块角色（无动画）。下一步需要引入真实的口袋妖怪风格视觉效果，让玩家能够在一个有村庄氛围的地图中自由探索，为后续添加 NPC、交互内容打好基础。

已有素材：`16x16RetroTileset.zip`，包含：
- `PokemonLike.png`：640x640 像素，40×40 格的 16x16 tileset
- 角色行走动画：5 个方向（Idle/上/下/左/右），每方向 4 帧，16x16 像素

## 功能描述

玩家打开网页，看到一个口袋妖怪风格的村庄地图，地图大于屏幕（约 2-3 倍屏幕大小），相机跟随玩家移动。玩家控制一个带行走动画的角色在地图上移动，遇到树木、房屋、水面等障碍物时无法穿过。

## 功能边界

### 本次包含
- 用代码（二维数组）定义村庄地图数据，加载 PokemonLike.png 渲染真实 tiles
- 村庄地图元素：草地（地面）、路径、房屋、树木、水面/池塘、围栏（选择性）
- 地图大小约为屏幕的 2-3 倍（≈ 1600×1200 像素），带世界边界
- Phaser 相机跟随玩家移动
- 替换蓝色方块为 CharacterAnimation 目录中的角色精灵
- 角色行走动画：4 方向（上/下/左/右），行走时播放对应方向动画，停止时播放 Idle 动画
- Tile 碰撞：障碍物 tiles（树木、房屋、水面、边界）阻挡玩家通行

### 本次不包含
- Tiled 编辑器 / 外部地图文件（.tmx/.json）
- NPC、对话系统
- 场景切换（室内/室外）
- 音效、背景音乐
- 地图内容说明（物品、信息看板）
- 玩家存档、坐标记录

## 验收标准

1. 地图使用 PokemonLike.png 中的 tiles 渲染，视觉上呈现口袋妖怪风格的村庄场景（有草地、路径、树木、至少一栋房屋）
2. 地图大小大于屏幕，相机跟随玩家移动，不超出地图边界
3. 玩家角色使用 CharacterAnimation 中的精灵，移动时播放对应方向的行走动画，停止时播放 Idle 动画
4. 玩家无法穿过障碍物 tiles（树木、房屋、水面）
5. 键盘方向键和虚拟 D-pad 均可正常控制角色移动
