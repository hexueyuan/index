# Pixel Portfolio

A pixel-art personal website built with Phaser 3 - explore an interactive village to learn more about me.

**Live Demo**: https://hexueyuan.github.io/index

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output will be in the `dist/` directory.

## Project Structure

```
src/                    # Source code
  scenes/               # Phaser scenes
  entities/             # Game entities
public/
  assets/
    tilesets/           # Terrain tiles
    sprites/            # Character/NPC sprites
    images/             # Other images
    audio/              # Sound effects (reserved)
docs/
  DESIGN.md             # Design decisions
  assets/               # Documentation assets
```

## Tech Stack

- **Phaser 3** - 2D game framework
- **Vite** - Build tool
- **TypeScript** - Language
- **GitHub Pages** - Hosting
