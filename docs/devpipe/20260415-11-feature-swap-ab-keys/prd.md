# 需求文档 (PRD)

## 基本信息

- 开发类型: 新功能
- 功能描述: 交换方向键与A键的左右位置，新增B键用于取消
- GitHub Issue: #11
- 远程分支: main
- 本地分支: feature-swap-ab-keys
- 创建时间: 2026-04-15

## 需求背景

当前移动端虚拟按键布局为：A键在左下角，方向键(D-Pad)在右下角。这与传统游戏手柄布局（左手方向键、右手动作按钮）相反，不符合玩家操作习惯。同时，游戏目前没有独立的"取消/返回"按钮，所有关闭操作都复用了A键，语义不够清晰。

## 功能描述

1. **交换按键位置**：将方向键(D-Pad)移到左下角，A键移到右下角，使布局符合传统游戏手柄习惯（左手操控方向，右手执行动作）。

2. **新增B键（取消）**：在右下角增加B键，与A键呈斜对角排列（A在右上，B在左下），用于关闭当前激活的UI元素。具体行为：
   - 对话框(DialogBox)激活时：直接关闭整个对话（不推进到下一页，区别于A键的推进行为）
   - 检查面板(InspectPanel)激活时：关闭面板
   - 无UI激活时：无效果

3. **PC端键盘映射**：B键功能在PC端映射到 Escape 键。

## 功能边界

### 本次包含
- 移动端虚拟按键位置交换（D-Pad 移到左下角，A键移到右下角）
- 新增移动端虚拟B键（圆形按钮，与A键斜对角排列）
- B键功能：关闭当前激活的对话框(DialogBox)或检查面板(InspectPanel)
- PC端 Escape 键绑定为B键（取消）功能

### 本次不包含
- 不改变A键的现有功能逻辑
- 不增加新的UI面板或对话内容
- 不改变PC端方向键和其他已有键盘绑定

## 验收标准

1. 移动端方向键显示在屏幕左下角，A键和B键显示在屏幕右下角
2. A键在右上、B键在左下，呈斜对角排列
3. 按下B键（移动端）或 Escape 键（PC端），能直接关闭当前显示的对话框（不推进对话，直接退出）
4. 按下B键（移动端）或 Escape 键（PC端），能关闭当前显示的检查面板
5. 无UI激活时，B键无效果（不触发交互）
6. A键的所有原有功能不受影响
